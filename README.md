# PHASOR MEASUREMENT UNIT (PMU)

## Secure 3-Phase PMU with Cyber Attack Detection

This project implements a smart and secure Phasor Measurement Unit (PMU) using ESP32.

## Features

- Real-time voltage measurement
- Current monitoring
- Frequency measurement
- Phase angle calculation
- GPS synchronization
- Cyber attack detection
- OLED display interface
- WiFi monitoring dashboard
- Google Sheets cloud logging
- Relay control system

## Hardware Components

- ESP32
- ACS712 Current Sensor
- ZMPT101B Voltage Sensor
- GPS Module
- OLED Display
- Relay Module

## Software Used

- Arduino IDE
- ESP32 Board Package
- TinyGPS++
- Adafruit SSD1306
- WebServer Library
