The modernization of power systems through smart grid integration has created an urgent need for low-cost, time-synchronized monitoring devices capable of detecting cyber threats targeting measurement infrastructure. This paper presents the design and implementation of a Phasor Measurement Unit (PMU) with embedded real-time cyber-attack detection, developed through MATLAB/Simulink simulation and physical hardware prototyping. The simulation model implements Discrete Fourier Transform (DFT)-based phasor estimation with 64 samples per cycle, extracting voltage magnitude, current magnitude, phase angle, frequency, and synchronized timestamps for three-phase power supply. The physical prototype uses an ESP32 microcontroller interfaced with ZMPT101B voltage sensors, ACS712 current sensors, ADS1115 16-bit ADC, and NEO-6M GPS module for microsecond-level time synchronization. The cyber-attack detection engine runs five parallel validation checks: Kirchhoff's Current Law verification, phase angle sum consistency (360° check), GPS distance monitoring for spoofing detection, rate-of-change analysis for impossible current spikes, and API key-based authentication for remote commands. Experimental validation demonstrates measurement accuracy of ±0.13% for voltage, ±0.61% for current, ±0.5° for phase angle, with cyber-attack detection achieving 97.1% accuracy and 22ms average latency across FDI attacks, GPS spoofing, replay attacks, and command injection attempts. The complete system operates at under 5 W with a hardware cost of approximately ₹4,000 – ₹10,000, making it suitable for research laboratories, educational institutions, and distribution-level grid monitoring applications.
